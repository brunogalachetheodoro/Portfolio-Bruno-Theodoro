Certificações

Técnologo Análise e Desenvolvimento de Sistemas

Formação superior completa na área de T.I. Pude aprender desde os conhecimentos básicos como Algoritmos e estruturas de dados aré tópicos mais avançados como Programação Orientada a Objeto.

JavaScript

Aprendi os primeiros conceitos de JavaScript como variáveis, tratamento de dados, operadores, DOM, condições, repetições, variáveis compostas e funções. Tudo treinado com exercícios práticos.

Git e GitHub

Aprendi nesse curso como foi criado o Git e o GitHub, a diferença entre eles e suas funções mais inportantes na prática como: Criar e clonar repositório, versionamento de código e criar commit através do GitHub desktop.

Algoritmo e Lógica de Programação

Aprendi nesse curso conceitos de lógica de programação e algoritmos como: estruturas condicionais e de repetição, funções, vetores e matrizes.

Figma do Zero ao Pro

Aprendi a criação de projetos de design para web, selecionando paleta de cores e fontes, focando na experiência do usuário. Tudo aplicado em projetos práticos.

HTML e CSS

Aprendi nesse curso que possui quatro módulos, todos os conceitos essenciais de HTML e CSS como a estruturação do HTML e sua semântica, a criação de formulários, estilização com CSS, e responsividade. Tudo isso estudado com projetos práticos.

CSS: Técnicas Avançadas

Aprendi nesse curso conceitos avançados de CSS como Flex-Box para o posicionamento dos elementos na página e responsividade. Conceitos esses aplicados na construção desta página como todos os projetos nela também.
