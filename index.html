Sobre Mim

Apaixonado por tecnologia, pude encontrar no desenvolvimento web uma forma de expressar minha criatividade e contribuir na vida de outras pessoas.

Trabalho em equipe,autodidata, liderança e capacidade de resolução são diferenciais que me destacam para agregar e contribuir com qualquer equipe que for trabalhar.

Como autodidata, pude aprender as habilidades técnicas necessárias e aplicá-las em projetos práticos simulando a realidade, criando projetos úteis.

Gamer, leitor e entusiasta esportivo, meus hobbies me mantém equilibrado para melhor exercer minhas tarefas como desenvolvedor e ainda me ajudam no desenvolvimento pessoal.

Certificações

Git e GitHub

Aprendi nesse curso como foi criado o Git e o GitHub, a difrença entre eles e suas funções mais importantes na prática como: Criar e clonar um repositório, versionamento de código e criar commit através do GitHub desktop.

Algoritmo e Lógica de Programação

Aprendi nesse curso conceitos de lógica de programação e algoritmos como: estruturas condicionais e de repetição, funções, vetores e matrizes.

Figma do Zero ao Pro

Aprendi a criação de projetos de design para web, selecionando paleta de cores e fontes, focando na experiência do usuário. Tudo aplicado em projetos práticos.

HTML e CSS

Aprendi nesse curso que possui quatro módulos, todos os conceitos essenciais de HTML e CSS como a estruturação do HTML e sua semântica, a criação de formulários, estilização com CSS, e responsividade. Tudo isso estudado com projetos práticos.
